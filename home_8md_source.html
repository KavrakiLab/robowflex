9 ## robowflex_library
10 A library that simplifies using [_MoveIt!_](https://github.com/ros-planning/moveit) in applications.
11 This is the core library used by all other components in Robowflex.
12 Take a look at the [design description](\ref design) for more information on the core library.
13 
14 ## Optional Components
15 
16 There are also a few optional modules that are compiled if the requisite libraries are available.
17 These will only compile if the necessary components are found.
18 
19 ### robowflex_ompl
20 An optionally compiled library component that adds more direct access to [OMPL](http://ompl.kavrakilab.org/) through a new `robowflex::Planner`.
21 Requires [`moveit_planners_ompl`](https://github.com/ros-planning/moveit/tree/kinetic-devel/moveit_planners/ompl), from [_MoveIt!_](https://github.com/ros-planning/moveit).
22 
23 ### robowflex_tesseract
24 An optionally compiled library component that adds support for [tesseract](https://github.com/ros-industrial-consortium/tesseract)-based planners.
25 Currently, only [OMPL](http://ompl.kavrakilab.org/)-based planning is available.
26 Requires both [tesseract](https://github.com/ros-industrial-consortium/tesseract) and [trajopt](https://github.com/ros-industrial-consortium/trajopt_ros).
27 
28 ### robowflex_movegroup
29 A library component with helper classes and functions to interact with a `move_group` process being used for motion planning.
30 Scenes can be pushed and pulled and trajectories can be executed with this component through `move_group`.

40 ### robowflex_doc
41 Documentation for all library and module components in robowflex (sans `tmpack`).
42 Documentation is automatically generated using [Doxygen](http://www.stack.nl/~dimitri/doxygen/), and is placed in `${CATKIN_DEVEL_PREFIX}/share/robowflex_doc/doc/index.html`.
43 The documentation can be read online at [GitHub pages](https://kavrakilab.github.io/robowflex/).
