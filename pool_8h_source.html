182  /** \brief Submit a function with arguments to be processed by the thread pool.
183  * Submitted functions must be wrapped with robowflex::make_function() or be a std::function type so
184  * argument template deduction works.
185  * \param[in] function Function to execute.
186  * \param[in] args Arguments to the function.
187  * \tparam RT Return type of function.
188  * \tparam Args Types of the arguments to the function.
189  * \return A job that contains information about the submitted function. This job can be canceled,
190  * which results in no execution of the function by the queue.
191  */

194  {

196  std::forward<Args>(args)...);
197 
198  {
